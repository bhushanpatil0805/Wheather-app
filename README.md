# 🌤️ Weather Today App

A simple and modern weather application built using **HTML, CSS, and JavaScript** that allows users to search for any city and view real-time weather information using the WeatherAPI service.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature display
- 🤗 Feels-like temperature
- 💨 Wind speed information
- 💧 Humidity details
- 🌤️ Weather condition icons
- 📱 Responsive design for mobile and desktop
- ⚡ Fast and lightweight application
- ⌨️ Press Enter key support for quick search

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- WeatherAPI
